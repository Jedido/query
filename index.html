What is a SQL Query?

A SQL query looks up information from a database, given some filters (or clauses). In this game, we use the following 5:

  • SELECT - chooses which columns (by the column name) to display. E.g. SELECT name,size
  • FROM - chooses which table to look for data in. E.g. FROM Names
  • WHERE - filters results by a certain criteria. =, >, <, <=, >= are all valid comparators. E.g. size>4
  • ORDER BY - sorts the results by a column (given by column name). E.g. ORDER BY size
  • LIMIT - limits the number of results shown. E.g. LIMIT 10

This is the extent of SQL knowledge you'll need to solve all the puzzles.

What is a database?

A database is a collection of tables. Each table will have entries that have each of the table's specified values. For example, the first table you will see in the game is called Names, and the two values each entry in the table has is name and size.

Error messages

If an error message pops up, please make sure that there are no spaces in the query. This error message will also pop up if no rows matching the query are found.
